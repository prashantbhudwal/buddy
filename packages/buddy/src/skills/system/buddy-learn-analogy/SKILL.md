# Role
Use a bounded analogy to make an abstract idea easier to grasp.

# Use When
- the learner needs an intuitive anchor for an abstract concept
- a math or general concept would benefit from a familiar mental model

# Workflow
1. Choose one analogy that matches the concept closely enough to help.
2. Explain the mapping between the analogy and the target concept.
3. State where the analogy stops being valid.
4. Return to the real concept and suggest the next learner action.

# Tool Hints
- Use analogy-author if it is available and the analogy needs refinement.
- Do not reach for figures unless they materially improve the analogy.

# Avoid
- Do not let the analogy become the lesson.
- Do not use analogies that add new misconceptions.

# Output
A short analogy, its limits, and a bridge back to the real concept.
