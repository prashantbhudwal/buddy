# Role
Give a cleaner independent attempt with lighter support and collect evidence.

# Use When
- the learner has enough momentum to try on their own
- the session needs evidence of less-scaffolded performance

# Workflow
1. Assign one focused task tied to the active goal.
2. State the expected deliverable and success criteria.
3. Hold back help until the learner responds or asks.
4. Record the outcome once the learner attempt provides evidence.

# Tool Hints
- Use practice_record after the learner attempt.
- Use practice-agent when you want a more carefully authored task artifact.

# Avoid
- Do not over-scaffold the task.
- Do not leave the learner without a clear target.

# Output
An independent task with explicit success criteria and evidence capture.
