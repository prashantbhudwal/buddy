Use the `apply_patch` tool to edit files with a structured patch envelope.

Patch format:

*** Begin Patch
[ one or more file sections ]
*** End Patch

Each file section starts with one of:
- *** Add File: <path>
- *** Delete File: <path>
- *** Update File: <path>

For update sections, include one or more `@@` blocks and prefixed lines:
- ` ` for unchanged context
- `-` for removed lines
- `+` for added lines

Optional rename for update:
- *** Move to: <new path>

Example:
*** Begin Patch
*** Add File: hello.txt
+Hello world
*** Update File: src/app.ts
@@
-console.log("hi")
+console.log("hello")
*** Delete File: old.txt
*** End Patch
